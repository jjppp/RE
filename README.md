# RE
简单的正则表达式识别器

# 原理
正则表达式->双栈处理得到后缀表达式->构造正则表达式的AST->遍历AST构造等价的NFA->根据NFA->构造等价的DFA

# 特色
成功识别正则表达式后将会导出`DFA.gv`和`NFA.gv`两个文件，用graphviz即可获得对应自动机的图例
